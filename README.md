### Операционная система UNIX
Робачевский Андрей Михайлович - https://books.google.ru/books?id=AdHDYdGKPuMC&printsec=frontcover&hl=ru#v=onepage&q&f=false

1. 1971г - 1ая версия UNIX: Появление первой версии UNIX.

2. 1982г - версия System 3: Разработка версии UNIX под названием System 3.

3. 1983г - версия System 5: Релиз версии UNIX под названием System 5.

4. 1984г - SVR2: Выпуск версии UNIX сокращенной как SVR2 (System V Release 2).

5. 1987г - SVR3: Разработка и выпуск версии UNIX сокращенной как SVR3 (System V Release 3).

6. 1989г - SVR4: Релиз версии UNIX под названием SVR4 (System V Release 4).

7. 1978г - BSD UNIX: Версия UNIX разработанная в университете Беркли, известная как BSD UNIX.

8. 1979г - 3BSD: Разработка третьей версии BSD UNIX.

9. 1983г - 4.2BSD с кучей багов: Выпуск версии 4.2BSD с большим количеством ошибок.

10. 1986г - 4.3BSD с фиксами и лучшей производительностью с NFS и VFS: Релиз версии 4.3BSD с исправлениями, включающими поддержку NFS и VFS, и улучшенной производительностью.

11. 1993г - 4.4BSD и BSD Lite: Выпуск версии 4.4BSD и BSD Lite.

12. 1988г - OSF/1: Разработка и выпуск операционной системы OSF/1, основанной на коде UNIX.
Микроядро Mach: Разработка и использование микроядерной архитектуры Mach для UNIX-подобных систем.
Микроядро Chorus: Создание систем на основе микроядра Chorus, включая Chorus/MiX v.4 и Chorus/MiX v.3, которые служат в качестве серверизации для систем SVR3 и SVR4.
